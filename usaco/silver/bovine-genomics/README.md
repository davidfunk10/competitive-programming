## Bovine Genomics (USACO Silver)

First USACO Silver problem! For this complete search problem, I created an array with 64 slots, enough for each of the 64 permuations of `XXX` where `X` can be `A`, `C`, `G`, or `T`. I could loop through the spotty cows, set the appropriate index of array `a` to 1 and then loop through the plain cows to see if they shared any of the permutations of XXX that the spotty cows had. If so, those three positions could not explain spottiness. I used a counter variable to count how many differnent 3 positions could genetically explain spottiness and outputted the answer. The main thing I learned from this problem was that I can sometimes create my own indexing system instead of using a set when the number of possible values is small and fixed to limit time complexity.
